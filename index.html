Predicting Facebook Friendships

Modeling how network structure in the Facebook friendship graph relates to the probability of future connections using collaborative filtering and machine learning techniques.

1. Predictive Task

The objective of this project is to predict whether two Facebook users are friends based solely on the structure of the social network. This is framed as a supervised binary classification problem, also known as link prediction.

Problem Formulation

For each candidate pair of users (u, v), we predict:

y = 1 → friendship exists y = 0 → no friendship Binary Classification

Input Features

  • Number of common friends (common neighbors)
  • Jaccard similarity of friend sets
  • Cosine similarity
  • Pearson correlation coefficient

Evaluation Metrics

  • ROC AUC - Ranking quality across all thresholds
  • Average Precision - Precision-recall summary
  • Accuracy - Classification correctness

Small-World Properties

  • Short Paths: Average distance < 4 hops
  • 90th Percentile: ≈ 5 hops

These properties confirm strong triadic closure - friends of friends are likely to be friends.

3. Feature Engineering

Collaborative Filtering Features

# For each pair (u, v): Γ(u) = set of neighbors of u Γ(v) = set of neighbors of v 1. Common Neighbors = |Γ(u) ∩ Γ(v)| 2. Jaccard = |Γ(u) ∩ Γ(v)| / |Γ(u) ∪ Γ(v)| 3. Cosine = |Γ(u) ∩ Γ(v)| / √(|Γ(u)| × |Γ(v)|) 4. Pearson = correlation coefficient

Dataset Construction

  • 20,000 positive examples (existing edges)
  • 20,000 negative examples (random non-edges)
  • 70/30 stratified train-test split
  • Balanced classes to avoid bias

6. Discussion

Main Findings

  • Simplicity Wins: Common Neighbors (1960s concept) matches state-of-the-art Gradient Boosting (2000s ML)
  • Zero Training Time: Heuristic requires no model training, enabling real-time predictions
  • Strong Triadic Closure: Friends-of-friends mechanism dominates Facebook network formation

Practical Implications

10,000x faster inference Perfect interpretability Scales to billions No training needed

Limitations & Future Work

  • Static snapshot - could explore temporal dynamics
  • Consider node attributes beyond structure
  • Test on other social networks
  • Explore Graph Neural Networks for comparison
